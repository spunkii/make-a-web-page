Cascading Style Sheets CSS

CSS is code written to control the "style" of HTML elements. You can have several files that can be used for your HTML file.
The most specific rule is applied to every element
Most specific rule applied is what cascading means, you keep going down until you find the one that best describes the elements, then that one gets applied.

To make a font style bold you use the font-weight property just as I did in the paragraph above.


You code it in CSS like this:

p.normal {
font-weight: normal;
}
p.light {
font-weight: lighter;
}
p.thick {
font-weight: bold;
}
p.thicker {
font-weight: 900;
}

You then use the class selector for your paragraph inside the p tag.

Box Model

Thinking in terms of structure is important because it alows the developer to easily provide the user a different experience. For instance, if you think of your page in boxes, big or small, you will find you can code them as such and then move the boxes around with ease.

Contents of the Box:

  • Content
    • Starting at the center of The Box Model you have content.
  • Padding
    • Clears the area around content and is affected by the background color of the box.
    • Space inside the box protets the content directly.
  • Border
    • Goes around the padding and content
  • Margin
    • Clears area around border, space inbetween boxes, no color

The sum of the element is computed by all the boxes:


1.) Padding
2.) Content
3.) Border
(Margin is not included in the size of the calculations when you sum the element size of the box.)


This is the CSS for sizing the box:

*{
-webkit
-moz
-ms box-sizing: border-box; }

You can set the overall box size and the padding and border will stay the same.
The above style is browser specific.
Box sizes are % or px. Pixels is defined by the pixel. The % is based off the site of the page.
